# 台灣潛點迷霧地圖 — 設計文件

> 潛水版「迷霧探索」：一張被海霧覆蓋的台灣潛點地圖，標記去過的潛點就散開迷霧，
> 累積潛水日誌（心得＋照片），用探索度與成就驅動「想去清下一塊霧」的循環。

## 一、核心概念

核心循環：**潛完水 → 回來點亮地圖 → 寫心得傳照片 → 看探索度上升 → 想去把下一塊霧清掉**

## 二、地圖與迷霧機制

- 底圖用 Leaflet + 海圖風格圖磚（深色海洋系），聚焦台灣本島＋離島
- 迷霧是一層 canvas 覆蓋：未解鎖區域呈半透明霧白／深灰，看得到輪廓但看不到潛點名稱
- 解鎖一個潛點 → 以該點為圓心的迷霧用羽化邊緣散開（1–2 秒擴散動畫）
- 未解鎖潛點在霧中只顯示「?」剪影標記，製造「那裡還有東西」的探索感
- 縮放到區域層級時，各區顯示進度環（例：東北角 8/21）
- 迷霧顆粒度：以潛點為單位散霧（每點清一小圈，區域全清才整片亮）

## 三、潛點資料庫（網路彙整內建 119 點，可自行新增）

| 區域 | 代表潛點 | 約略數量 |
|------|---------|---------|
| 東北角 | 龍洞灣、和美、鼻頭角、卯澳、馬崗 | ~20 |
| 墾丁 | 後壁湖、出水口、合界、萬里桐、佳樂水 | ~15 |
| 小琉球 | 美人洞、花瓶岩、厚石裙礁、杉福 | ~10 |
| 綠島 | 石朗、柴口、大白沙、鋼鐵礁 | ~12 |
| 蘭嶼 | 八代灣、玉女岩、雙獅岩、開元港 | ~10 |
| 澎湖 | 東吉、西吉、南方四島 | ~10 |
| 其他 | 龜山島、基隆嶼、台東杉原、花蓮石梯坪 | ~8 |

每個潛點固定屬性：座標、難度（入門／進階／技潛）、類型（岸潛／船潛）、
最大深度、特色（沉船、軟珊瑚、海龜熱點…）。找不到的點可長按地圖新增。

## 四、解鎖與日誌流程

點潛點 →「標記已潛」→ 日誌卡片：

- **基本**：日期、潛伴/潛店、氣瓶支數
- **數據**（選填）：最大深度、潛水時間、水溫、能見度
- **生物標籤**：快速勾選 tag（海龜、梭魚風暴、豆丁海馬…）
- **心得**：自由文字
- **照片**：多張上傳，前端壓縮後存 IndexedDB，點開看大圖
- 同一潛點可累積多筆紀錄，地圖標記顯示潛次數

## 五、遊戲化元素

- **探索度**：全台總 % ＋ 各區域 %，首頁大字顯示
- **徽章成就**（15–20 個）：開光首潛、東北角制霸、離島收集者（四離島全解鎖）、
  深藍勳章（30m+）、夜行者（夜潛）、百潛俱樂部…
- **統計頁**：總潛次、總時數、最深紀錄、最常去的點、逐月潛水熱力圖（GitHub 草皮風格）
- **生物圖鑑**（Phase 2）：勾過的生物標籤累積成圖鑑，未見過的呈剪影

## 六、技術架構

- **單檔 HTML**（index.html），Leaflet 走 CDN
- **儲存**：紀錄用 localStorage、照片壓縮後存 IndexedDB，一鍵匯出／匯入 JSON 備份
- **RWD**：手機直式為主要使用情境，地圖全螢幕、日誌卡片由下往上滑出
- **部署**：GitHub Pages（repo 轉 public 後開啟）

## 七、開發階段

- **Phase 1（MVP）**：地圖＋迷霧解鎖動畫、內建潛點資料庫、日誌（心得＋照片）、
  區域進度、匯出備份
- **Phase 2**：徽章成就、統計頁（熱力圖）、生物圖鑑、自訂潛點
- **Phase 3（可選）**：Firebase 同步跨裝置、分享探索地圖成圖片（潛水員名片）

## 八、已定案的決策

1. 儲存：先純本機（localStorage + IndexedDB，免帳號、離線可用），Phase 3 再考慮 Firebase
2. 專案位置：獨立 repo（taiwan-dive-map），不併入外層 bill-split
3. 迷霧顆粒度：以潛點為單位散霧
