# 台灣潛點迷霧地圖 (Taiwan Dive Fog Map)

潛水版迷霧探索地圖：標記去過的台灣潛點解鎖迷霧，累積潛水日誌（心得＋照片），
用探索度與成就驅動探索循環。

- 設計文件：[DESIGN.md](DESIGN.md)
- 主程式：`index.html`（單檔 HTML，開瀏覽器即用）
- 資料全存本機（localStorage + IndexedDB），支援 JSON 匯出備份

## 開發狀態

- [x] Phase 1 (MVP)：地圖＋迷霧解鎖、內建潛點庫（網路彙整全台 119 點）、日誌（心得＋照片）、區域進度、匯出/匯入備份、長按新增自訂潛點
- [x] Phase 1.5：水肺 🤿 / 自潛 🐬 雙分支（獨立迷霧、日誌、探索度）、PADI / Subsurface 日誌匯入（中英文潛點名稱自動比對＋手動指定＋去重）
- [x] Phase 1.6：七日海況預報（Open-Meteo 浪高/風速/水溫＋紅黃綠燈）、願望清單 ⭐（星標＋篩選＋週末海況燈）、探索分享圖 📸、潛點裝備備忘 🎒
- [x] Phase 1.7：潛點清單 🔍（依區域瀏覽＋中英文搜尋＋狀態篩選；點清單任一潛點會在地圖脈動定位，確認方位）、GitHub Pages 自動部署
- [x] Phase 1.8：離線可用（Service Worker 快取本體＋地圖＋Leaflet，可加入主畫面成 PWA）、日誌可編輯（改欄位／增刪照片）、密集潛點地圖聚合（放大才散開）
- [x] Phase 1.9：🛰️ 衛星圖層切換（偵查礁石與入水點，切換時迷霧暫退）、✥ 校正潛點座標（拖曳金色標記把概略座標移到正確位置並記住，可還原）
- [ ] Phase 2：成就徽章、統計熱力圖、生物圖鑑
- [ ] Phase 3：Firebase 同步、探索地圖分享圖
